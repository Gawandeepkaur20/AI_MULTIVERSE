# 📖 Overview

AI Multiverse is an intelligent multi-tool AI platform built with **Streamlit** that combines:

- 🤖 Multi-Personality AI Chat
- 🎨 AI Image Generation
- 🧠 Stateful Conversations
- 🌍 Translation Support
- 📊 Chat Analytics
- ⚡ Beautiful Modern UI

Instead of a single chatbot, users can switch between different AI personalities, each providing responses in its own unique style.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI_Multiverse.git
```

Move into the project

```bash
cd AI_Multiverse
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env`

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run

```bash
streamlit run app.py
```
